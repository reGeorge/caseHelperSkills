# 飞书表格字段映射

## 飞书表格字段 → SDET用例字段

| 飞书字段 | SDET字段 | 说明 |
|---------|---------|------|
| 用例名称 | name | 用例名称 |
| 用例编号 | note | 作为备注信息的一部分 |
| 用例描述 | description | 用例描述 |
| 预置条件 | - | 不映射到SDET |
| 测试步骤 | - | 不映射到SDET（公共步骤替代） |
| 期望结果 | - | 不映射到SDET |
| 是否支持自动化 | - | 用于筛选，不映射 |
| 脚本序号 | - | 用于筛选，不映射 |

## 筛选逻辑

### 条件1: 是否支持自动化
- **值**: "是"
- **其他值**: "否"、null、空字符串等都会被过滤

### 条件2: 脚本序号
- **空值**: null、空字符串
- **有值**: 任何非空值（如66164）都会被过滤

### 组合条件
```python
if case.get("是否支持自动化") == "是" and not case.get("脚本序号"):
    # 符合条件
```

## 用例创建字段构造

### 必填字段
```python
{
    "productId": 1,  # 固定值
    "name": case["用例名称"],
    "caseType": 2,  # 固定为自动化用例
    "parent": 66241,  # 目标目录ID
    "creator": "自动化脚本",
    "creatorId": "9999",
    "modifier": "自动化脚本",
    "modifierId": "9999",
    "status": 0,
    "deleted": 0
}
```

### 可选字段
```python
{
    "priority": 2,  # 默认优先级
    "note": f"来源: {case['用例编号']}",
    "description": case.get("用例描述", "")
}
```

## 公共步骤映射

公共步骤从用例51401复制，字段映射：

| 源字段 | 目标字段 | 说明 |
|-------|---------|------|
| name | name | 保持原样 |
| order | order | 保持原样 |
| host | host | 通常为${G_host} |
| protocol | protocol | 0=HTTP, 1=HTTPS |
| path | path | 请求路径 |
| method | method | HTTP方法 |
| headers | headers | 请求头 |
| body | body | 请求体（JSON字符串） |
| params | params | 查询参数 |
| variables | variables | 步骤变量 |
| check | check | 断言列表 |

**重要**: 需要将`caseId`修改为新创建的用例ID。

## 注意事项

1. **字段为null的处理**: 使用`.get()`方法，提供默认值
2. **特殊字符**: 用例名称中的特殊字符保持原样
3. **换行符**: 预置条件等字段中的换行符`\n`保持原样
4. **空字段**: 飞书表格中空字段可能返回null或空字符串
5. **mention类型**: 飞书表格中的@提及会被解析为复杂对象，需提取text字段
